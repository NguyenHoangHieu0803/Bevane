# Bevane
1. This is the demo app using the Agent Teams to build the private message app. 

The target of this app is to create a system which IOS devices can scan the QR code for downloading the app, connect to other device using the same app and able to communicate to each other via chat, video call or phone call. AI also applied in this application.

Application name: Bevane.
2. Requirement:
- There are 4 main function:
-- Messaging.
-- Calling.
-- Video calling.
-- Note.

To implement this application, there will be 4 agents doing this task:
- Business Analyst.
- Back End developer.
- Front End developer.
- QA/QC Full-stacks.
